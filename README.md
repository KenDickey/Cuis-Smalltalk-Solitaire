Cuis-Solitaire
==========
Tested with Cuis 4.2 rev 2226

Solitaire:  Klondike & FreeCell Solitaire Games

````Smalltalk
	Feature require: 'Morphic-Games-Solitaire'
````

Then execute

````Smalltalk
    Klondike newGame.
    FreeCell newGame.
````

Or use the World Menu to invoke:  New Morph > Games-Solitaire > FreeCell
